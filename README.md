# Edu Bootcamp Project (MySQL Project)

## Overview
This project is a MySQL database designed for a global coding bootcamp platform.

## Features
- Learner account management (registration, subscription, blocking)
- Bootcamp creation and mentor assignment
- Enrollment system with capacity checks
- Module progress tracking
- Assessments and results tracking
- Resource management with storage limits
- Error logging system

## Database Components
- Tables with constraints (PK, FK, UNIQUE, CHECK)
- Views (Top coders, blocked learners, etc.)
- Stored Procedures
- Triggers
- Indexes

## Tools Used
- MySQL Workbench
